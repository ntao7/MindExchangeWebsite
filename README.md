# 🧠 Mind Exchange Website

Welcome to the **Mind Exchange Plaza** — a simple web project that displays interesting facts across different categories such as technology and society.

---

## 🌐 Project Overview

This website demonstrates a lightweight front-end project structure built with **HTML** and **JavaScript**, featuring:
- A clean and semantic HTML layout (`index.html`)
- A modular JavaScript data file (`data.js`)
- A logo image (`logo.png`)
- Facts displayed in categories such as *technology* and *society*
